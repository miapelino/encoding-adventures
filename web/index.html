Text Encoder and Decoder

Guess what I learned this week?

In JavaScript, there's only one data type for a number.

So whether it's an integer, short, long, floating point number, doesn't matter. Apparently JavaScript numbers are always 64-bit floating point.


Which means that my beautiful integer encoding device below is only accurate up to 15 digits, whether I use the built in javascript function to convert, or my own function.
